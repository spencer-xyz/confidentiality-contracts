# Hardhat Project Template

Use this template to start building your projects on COTI using [Hardhat](https://hardhat.org/)

## Usage

1. Install dependencies
  ```
  yarn
  ```

2. Build and compile contracts
  ```
  yarn build
  ```

3. Run tests
  ```
  yarn test
  ```
